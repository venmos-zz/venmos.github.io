今天从Github上闲逛,偶尔发现一个养眼D主题, [[https://github.com/ChrisKempson/Tomorrow-Theme][Tomorrow Theme]].

这个主题的支持范围非常广泛,甚至包括OS X Terminal…:

- TextMate
- Vim (Full-colour and 256-colour)
- GNU Emacs
- Xcode 4
- Coda
- Eclipse
- Sublime Text 2 (Uses TextMate Themes)
- Redcar (Uses TextMate Themes)
- E TextEditor (Uses TextMate Themes)
- SubEthaEdit (Uses Coda Themes)
- BBEdit
- Ace Editor
- IntelliJ
- Visual Studio
- Notepad++
- iTerm2
- OS X Terminal
