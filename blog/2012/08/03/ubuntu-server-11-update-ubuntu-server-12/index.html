升级 Ubuntu Server 11 到 Ubuntu Server 12

今天下午台风来了,下班后被困在公司了,囧. 呆在公司无所事事,就想顺便把公司几台运行Ubuntu Server 11的服务器给系统升级到12算了.

步骤相当简单,首先更新所有现有软件到最新版本: 

$sudo apt-get update
$sudo apt-get upgrade

之后根据自己服务器的情况适当备份各种配置文件,之后运行系统版本升级: 

$sudo apt-get install update-manager-core
$sudo do-release-upgrade

完了之后,回答几个问题,就会开始自动升级了,升级结束后可以使用命令lsb_release -a来查看系统版本.
