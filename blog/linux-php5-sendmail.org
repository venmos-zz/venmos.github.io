安装Sendmail：

#+begin_src
apt-get install sendmail
#+end_src

编辑PHP.ini：

#+begin_src
emacs /etc/php5/apache2/php.ini
#+end_src

在PHP中的Sendmail参数后加入Sendmail路径：

#+begin_src
sendmail_path = /usr/sbin/sendmail
#+end_src
