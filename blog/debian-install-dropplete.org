Dropplets资源:

- Website http://dropplets.com
- Github https://github.com/Circa75/dropplets

安装Dropplets:

在Debian/Ubuntu上部署Apache2和PHP5:

#+begin_src
#apt-get install apache2 php5 php5-gd
#+end_src

从Dropplets的站点拖回压缩包,上传解压到VPS的站点根目录.

为Apache2开启.htaccess支持:

#+begin_src
#a2enmod rewrite
#+end_src

编辑sites-enabled里的站点配置文件,将所有AllowOverride的值修改为All之后重启Apache2服务.

#+begin_src
#/etc/init.d/apache2 restart
#+end_src
